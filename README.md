## Installing
```html
<script src="/path/to/easyDownload.min.js"></script>
```

## Usage
```html
<a id="download" href="javascript:void(0)" onclick="easyDownload.run('http://example.com/path/to/filename.pdf', 'pdf', 'something')">download</a>
```
